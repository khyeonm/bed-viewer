# bed-viewer

Interactive BED file viewer with column detection, color-coded fields, sorting, filtering, and region statistics.

## Features

- Auto-detects BED format (BED3 through BED12)
- Column header sorting (click to sort ascending/descending)
- Text search across all fields
- Chromosome dropdown filter
- Score visualization with mini bar charts
- Strand color coding (+ green, - red)
- Computed region length column
- Summary statistics (region count, chromosomes, avg/min/max length)
- Pagination for large files (100 rows per page)

## Supported Extensions

- `.bed`

## Installation

Install from the **Plugins** tab in the AutoPipe desktop app.
